Job Application Form

A simple HTML/CSS Job Application Form that allows users to submit their personal information, select a job position, choose availability, and send a message.

Features

Input field for full name

Input field for email address

Dropdown menu to select a job position (Frontend Developer, Backend Developer, UI/UX Designer, Project Manager)

Radio buttons to choose availability (Full-Time or Part-Time)

Textarea for additional message

Submit button

Technologies Used

HTML5 – for structure of the form

CSS3 – for styling (external stylesheet styles.css)

Responsive design using <meta name="viewport"> for mobile compatibility
